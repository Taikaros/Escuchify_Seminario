¿Qué es Escuchify?

Es un sistema Full Stack moderno diseñado para administrar bibliotecas de música de forma visual e intuitiva.

Gestión de Datos

Control total de Artistas, Discos y Canciones.

Automatización

Búsqueda automática de imágenes y metadatos.

Escalabilidad

Arquitectura desacoplada y robusta.

Stack Tecnológico

.NET 9

Web API Backend

Blazor

Interactive Frontend

Tailwind

Diseño Moderno

xUnit

Tests Unitarios


+ SQLite (Base de Datos) & Entity Framework Core

Clean Architecture

Decidimos separar las responsabilidades para evitar código "espagueti".

Frontend (Blazor App)
API (Controllers & DTOs)
Services & Data (Lógica & BD)
Core (Entidades Puras)

Funcionalidades Clave

  • CRUD Completo: Crear, Leer, Editar, Borrar.
  • Borrado en Cascada: Integridad de datos asegurada.
  • API Externa: Integración con Deezer/iTunes.
  • Validaciones: Control de errores en tiempo real.
  • Calidad: 100% Tests Unitarios aprobados.
